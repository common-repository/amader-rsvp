== Description ==

Amader RSVP is an amazing wordpress plugin for RSVP. You can add a form anywhere in your website, then your guests can submit their information through the RSVP form very easily.


**Public Demo Here:** <a href="https://mucasoft.com/plugins/amader-rsvp-wordpress-plugin-demo/" rel="friend"> https://mucasoft.com/plugins/amader-rsvp-wordpress-plugin-demo/</a>

== Installation ==

1. Upload 'amader-rsvp' directory to the "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress
3. Use [amader_rsvp] in any page, post or widget to show RSVP form in your website.


== How to use: ==

1. Simply add [amader_rsvp] in any page to show the RSVP form in your website.
2. Go to wordpress admin panel > Amader RSVP page to see your guests confirmations.
